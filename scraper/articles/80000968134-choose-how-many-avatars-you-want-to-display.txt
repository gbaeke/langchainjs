By default we don't display all avatars, but only 50% (0.5) of them. In case you want to display more or less avatars, you need to change the avatar fill percentage under your room settings. Go to the dashboardOpen the event in which you want to change the avatar fill percentageGo to the room in which you want to change the avatar fill percentageOpen the visual editorSelect 'room settings'Select 'avatar positions' Fill in perferred percentage under 'avatar fill percentage'  Needs to be a number between 0 and 1 0 = no avatars1 = all avatars
