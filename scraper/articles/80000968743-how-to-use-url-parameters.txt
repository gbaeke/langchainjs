URL parameters can be used to pass a value of the profile data in a link to an external application/website.For example: You want to link to an external platform directly into a user specific context. https://myapp.com/userid=12346789 Search in google for the jobtitle of the attendee: https://www.google.com/search?q=analist How to configure the URL parameters Go to dashboardOpen the event to which you want to add URL parametersOpen the room to which you want to add URL parameters  in the visual editorConfigure a button or emedded element by filling in the following:Link/slug: add {} with the parameter key in between to create a placeholder where the selected profile value will be replacedUrl/Query ParametersPlaceholder key: select the key that has been identified in the urlProfile field key: select the profile field that will be used to replace the placeholder
